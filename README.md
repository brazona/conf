# Configuration Server

Servidor de Configuração dedicado a armazenar arquivos de configurações das aplicações e de oferecer serviço rest para disponibilizar os dados.

``` text

│      └── deploy.yml
├── app
│   └── profile
│   └── service
├── docs
│   └── CONTRIBUTING.md
│   └── CODE_OF_CONDUCT.md
│   └── PULL_REQUEST_TEMPLATE.md
└── README.md
```
